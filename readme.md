## Система Service Deck для Онлайн-школы (Регистрация, запись, сопровождение заявок от студентов на онлайн-обучение)

Доступные методы API

- [Регистрация новой заявки на обучение (POST) http://online-school.phpecommerce.ru/api/bids/create](http://online-school.phpecommerce.ru/api/bids/create)
- [Постановка вновь созданной  заявки в очередь прозвонки (PATCH) http://online-school.phpecommerce.ru/api/bids/call/1](http://online-school.phpecommerce.ru/api/bids/call/1)
- [Предварительное подтверждение вновь созданной  заявки (PATCH) http://online-school.phpecommerce.ru/api/bids/accept/1](http://online-school.phpecommerce.ru/api/bids/accept/1)
- [Отклонение вновь созданной  заявки (PATCH) http://online-school.phpecommerce.ru/api/bids/reject/1](http://online-school.phpecommerce.ru/api/bids/reject/1)
- [Постановка вновь созданной  заявки в очередь ожидания (PATCH) http://online-school.phpecommerce.ru/api/bids/postponed/1](http://online-school.phpecommerce.ru/api/bids/postponed/1)
- [Подтверждение оплаты вновь созданной  заявки (PATCH) http://online-school.phpecommerce.ru/api/bids/confirm/1](http://online-school.phpecommerce.ru/api/bids/confirm/1)
- [Просмотр всех заявок на онлайн-обучение (GET) http://online-school.phpecommerce.ru/api/bids/list](http://online-school.phpecommerce.ru/api/bids/list)
- [Просмотр всех заявок на онлайн-обучение, ожидающих обработки (GET) http://online-school.phpecommerce.ru/api/bids/list/waiting](http://online-school.phpecommerce.ru/api/bids/list/waiting)
- [Просмотр всех принятых заявок на онлайн-обучение (GET) http://online-school.phpecommerce.ru/api/bids/list/accepted](http://online-school.phpecommerce.ru/api/bids/list/accepted)
- [Просмотр всех заявок на онлайн-обучение, находящихся в очереди прозвонки (GET) http://online-school.phpecommerce.ru/api/bids/list/called](http://online-school.phpecommerce.ru/api/bids/list/called)
- [Просмотр всех отклоненных заявок на онлайн-обучение (GET) http://online-school.phpecommerce.ru/api/bids/list/rejected](http://online-school.phpecommerce.ru/api/bids/list/rejected)
- [Просмотр всех заявок на онлайн-обучение, поставленных в очередь ожидания (GET) http://online-school.phpecommerce.ru/api/bids/list/postponed](http://online-school.phpecommerce.ru/api/bids/list/postponed)
- [Просмотр всех заявок на онлайн-обучение с подтвержденной оплатой (GET) http://online-school.phpecommerce.ru/api/bids/list/confirmed](http://online-school.phpecommerce.ru/api/bids/list/confirmed)


## Powered by Symfony 5
